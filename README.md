# AdminCore
Un AdminPanel versátil para Roblox.

## Caracteristicas
- Sistema de ban
- Sistema de kick
- Sistema de freeze
- Sistema de speedhack
- Sistema de jumphack
- Temas:
  - Liquid Glass (Puede tener errores)
  - Blur
- Ejecutar codigo con privilegios de Servidor

## Requerimientos
- Game Settings -> Allow Studio To Access API Services -> true
- ServerScriptService -> loadstring enabled -> true (para ejecutar codigo desde el admin panel)

> Efectos como blur o liquid glass requieren una calidad grafica mayor o igual a 8.
